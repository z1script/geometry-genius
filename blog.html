1. What is DOM and what is the purpose of DOM?

Ans: DOM is Document Object Model & The Purpose Of Dom Is To Manipulate Web Pages By Using Programming Language JavaScript


2. How will you select HTML elements using DOM? Name the DOM methods.

Ans: In DOM, There Are 5 Seletors. By Using Any Of The Selectors I Can Select HTML Elements.
Names Of The DOM methods Are Given Below:-
1. getElementById()
2. getElementsByClassName()
3. getElementsByTagName()
4. querySelector()
5. querySelectorAll()


3. What is event bubble?

Ans: Event Bubbling Is a Method of The HTML DOM & It Propagate DOM elements & To stop event Propagation You Can call a function named stopPropagation()
